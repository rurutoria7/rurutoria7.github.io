Hexo 初識

建立 hexo 專案：

或是把當前資料夾當成目錄：

Hexo 會射一坨檔案進去，（常用）目錄結構列舉如下：

  • scaffolds：放模板
  • source：放 post（文章）和 page（如 about、tag）
  • themes：主題直接 clone 到裡面
  • config.yml：站點配置文件

注意，主題配置文件也叫 config.yml，別搞混了。
